# Travel Insurance Front

旅遊險專案前端，使用 Vue3 + Quasar + TypeScript 開發。

## 環境需求

- Node.js v18 以上
- npm v9 以上

## 安裝步驟

```bash
# 1. clone 專案
git clone https://github.com/你的帳號/travel-insurance-front.git

# 2. 進入資料夾
cd travel-insurance-front

# 3. 安裝套件
npm install

# 4. 啟動開發伺服器
npm run dev
```

## 專案結構

```
src/
├── assets/       # 靜態資源
├── components/   # 共用元件
├── pages/        # 頁面
├── router/       # 路由設定
├── stores/       # Pinia 狀態管理
└── App.vue       # 根元件
```

## 主要套件

- Vue 3
- Quasar
- Vue Router
- Pinia
- TypeScript
